# Security Policy

## Supported Versions

Currently, we only support the latest version of BorraCord.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of BorraCord seriously. If you believe you have found a security vulnerability, please report it responsibly.

### How to report
Since this is an open-source project maintained by an individual, please report vulnerabilities by opening a new issue in the **[GitHub Issues](https://github.com/narc0tics4life/BorraCord/issues)** tab.

1.  **Do not** include sensitive personal data or valid Discord tokens in your report.
2.  Describe the vulnerability and provide steps to reproduce it.
3.  We will try to address the issue as soon as possible in a future update.

Thank you for helping keep BorraCord safe.
